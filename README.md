# ISMAUK Stress Management Awards 2026

[![GitHub Pages](https://img.shields.io/badge/GitHub-Pages-blue)](https://github.com/ISMAUK/ismauk-awards-2026)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

Online application portal for the ISMAUK Stress Management Awards 2026.

## 🏆 About the Awards

The ISMAUK Stress Management Awards recognise excellence in workplace wellbeing, psychological safety and stress reduction. They celebrate organisations, teams and individuals who demonstrate outstanding leadership, innovation and impact in reducing stress and promoting mental wellbeing at work.

**Established in 2024 to mark ISMAUK's 50th anniversary**, these awards highlight those who are shaping healthier, more productive and psychologically safe working environments.

### Award Tiers

- **🥉 Bronze Award** — Starting Out  
  Progress, not perfection. For early-stage commitment.

- **🥈 Silver Award** — Established Practice  
  Embedded practice & maturity. For well-established programmes.

- **🥇 Gold Award** — Strategic Excellence  
  Excellence, not just activity. For strategic integration.

**Deadline: 30 September 2026**

## ✨ Features

✅ **Interactive 4-Step Application Form**
- Award tier and applicant type selection
- Organisation/individual details
- Award-specific questions with guidance
- Review and PDF download

✅ **Complete Guidance**
- Detailed award descriptions with eligibility criteria
- Scoring rubrics for each tier
- Judges' notes for every question
- What makes a strong application (modal popup)
- Common pitfalls to avoid

✅ **Dynamic Question Loading**
- Questions adapt based on award tier selected
- Different questions for individuals vs. organisations
- Word count tracking and character limits
- Helper text that disappears as user types

✅ **PDF Generation**
- Applicants download completed application as PDF
- Professional formatting
- Ready to email to awards team
- No data stored on servers

✅ **Responsive Design**
- Works on desktop, tablet, and mobile
- Professional styling with ISMAUK brand colors
- Smooth animations and transitions

✅ **No Backend Required**
- Single HTML file
- No database needed
- All data stays in user's browser
- Completely private and secure

## 🚀 Quick Start

### Option 1: GitHub Pages (Easiest)

1. Fork this repository
2. Go to **Settings** → **Pages**
3. Set **Source** to `main` branch
4. Your site will be live at `https://yourusername.github.io/ismauk-awards-2026`

### Option 2: Local Testing

1. Clone the repository:
```bash
git clone https://github.com/yourusername/ismauk-awards-2026.git
```

2. Open `index.html` in your web browser (no server needed)

### Option 3: Self-Hosted

1. Download all files
2. Upload to your web server
3. Ensure `index.html` is the default page
4. Visit your domain

## 🎯 How It Works

1. **Applicants visit the portal**
2. **Read about the awards** (3 tiers with eligibility)
3. **Select award tier** (Bronze, Silver, or Gold)
4. **Choose applicant type** (Individual or Organisation)
5. **Complete 4-step form** with guidance
6. **Download PDF** of completed application
7. **Email PDF** to admin@ismauk.org.uk

## 📊 Application Flow

```
Landing Page
    ↓
Award Information (with scoring rubrics)
    ↓
4-Step Application Form
├─ Step 1: Category Selection
├─ Step 2: Your Details
├─ Step 3: Questions (dynamically loaded)
```

## 🎨 Customization

To customize the portal, edit `index.html`:

### Change Email Address
Find: `admin@ismauk.org.uk`  
Replace with: your email address

### Change Deadline
Find: `30 September 2026`  
Replace with: your deadline date

### Change Colors
In the `<style>` section, update CSS variables:
```css
--primary: #003d7a        /* Navy blue */
--accent: #a91e6a         /* Magenta */
```

### Update Award Criteria
Edit the award tier descriptions and modify question text as needed
